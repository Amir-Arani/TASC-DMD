Method

The TASC-DMD Algorithm

TASC-DMD transforms high-dimensional spatiotemporal data into a compact latent space using time-delay embedding and spatial compression, then applies Dynamic Mode Decomposition for spectral analysis.

01

Time-Augmented Embedding

Construct a 3D Hankel tensor by stacking d delay-shifted snapshot windows, capturing slow-time dynamics invisible to single-snapshot DMD.

02

Space-Contracted Projection

Compress each delay-level slice via a rank-r₀ SVD and stack into a reduced Hankel matrix of size (r₀·d) × (m−d+1).

03

DMD in Latent Space

Apply Exact DMD (or fbDMD) on the contracted matrix at rank r₁ to obtain eigenvalues and modes, then map back to physical space.

04

TASC-SINDy Extension

Feed low-dimensional TASC coordinates into SINDy to discover sparse governing equations from spatiotemporal measurement data.

Additional Example — Not in Paper

Advection-Diffusion Benchmark

An additional benchmark using the linear advection-diffusion equation with analytically known eigenvalues. This example is not included in the paper and is provided here as a standalone validation that allows direct comparison of each DMD method against exact ground truth.

✓ Reproducible  ·  Not in Paper  ·  MATLAB

Known Eigenvalue Parabola at 50% Noise

The exact eigenvalues lie on an analytically derived parabola in the complex plane. This parabola is overlaid on the DMD eigenvalue plot as ground truth — any deviation is directly attributable to noise sensitivity. Tested at 50% Gaussian noise. Cv is reported for all three methods.

u_t + U₀·u_x = ν·u_xx   |   ω_n = −ν·k_n² − i·U₀·k_n   |   Re(ω) = −(ν/U₀²)·Im(ω)²

Code

Repository Contents

All code is written in MATLAB. Core functions are in src/; benchmarks are in examples/.

DMD_exact.m
Exact DMD with time-delay embedding and Cv metric.
fbDMD.m
Forward-Backward DMD — noise-robust via geometric mean.
nls_rhs.m
ODE RHS for the NLS benchmark (Fourier space).
generate_NLS_data.m
Generates NLS two-soliton benchmark data. (paper)
Schrodinger_equ_TASCDMD.m
NLS benchmark — reproduces paper results. (paper)
generate_AdvDiff_data.m
Generates advection-diffusion data with known eigenvalues. Not in paper.
AdvDiff_TASCDMD.m
AdvDiff benchmark at 50% noise vs analytical parabola. Not in paper.

Quick Start

% Add functions to path
addpath('src');
cd examples

% NLS benchmark (paper)
run generate_NLS_data.m
run Schrodinger_equ_TASCDMD.m

% Advection-diffusion benchmark (NOT in paper)
run generate_AdvDiff_data.m
run AdvDiff_TASCDMD.m
