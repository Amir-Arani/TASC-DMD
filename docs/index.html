Abstract

Overview

Low-dimensional coherent patterns underlie the behavior of many complex dynamical systems. We propose a novel dynamic mode decomposition (DMD) framework to discover reduced-order models of complex physics from spatiotemporal data. This algorithm combines time-delay embedding and reduced-order spectral projections to transform the initial dynamics into a latent space where DMD can efficiently resolve their evolution in space and time.

Here, the efficiency and accuracy of this approach — which we call Time-Augmented, Space-Contracted DMD (TASC-DMD) — is demonstrated in several benchmark tests, and is then utilized to compactly characterize human brain deformation. Sparse Identification of Nonlinear Dynamics (SINDy) is then employed to discover a parsimonious model for brain deformation in TASC coordinates. This integrated algorithm (TASC-SINDy) is trained on 4D strain data from in vivo tagged MRI in 36 human subjects. A unique and sparse set of governing equations describing the temporal dynamics of the brain was discovered using only three generic modes.
Method

The TASC-DMD Algorithm

TASC-DMD operates in three conceptual stages that transform high-dimensional spatiotemporal data into a compact latent space before performing spectral decomposition.

01

Time-Augmented Embedding

Construct a 3D Hankel tensor by stacking d delay-shifted snapshot windows, capturing slow-time dynamics that are invisible to single-snapshot DMD.

02

Space-Contracted Projection

Compress each delay-level slice via a rank-r₀ SVD and stack the projected coordinates into a reduced Hankel matrix of size (r₀·d) × (m−d+1) — drastically reducing the state dimension.

03

DMD in Latent Space

Apply Exact DMD (or fbDMD) on the contracted matrix using rank r₁ to obtain eigenvalues, modes, and temporal dynamics — then map modes back to physical space via the stored spatial bases.

04

TASC-SINDy (Brain Application)

Feed the low-dimensional TASC coordinates into SINDy to discover a sparse, interpretable governing equation for the temporal brain dynamics from 4D tagged MRI strain data.

Code

Repository Contents

All code is written in MATLAB and organized into reusable functions (src/) and runnable examples (examples/).

DMD_exact.m
Exact (or standard) DMD with optional time-delay embedding and Cv metric.
fbDMD.m
Forward-Backward DMD — noise-robust via geometric mean of forward/backward operators.
nls_rhs.m
ODE right-hand side for the NLS equation in Fourier space (for ode45).
generate_NLS_data.m
Solves the NLS equation via split-step Fourier and saves benchmark data.
Schrodinger_equ_TASCDMD.m
Main benchmark script — compares TASC-DMD, Exact DMD, and fbDMD.

Quick Start

% 1. Add src/ to path and generate benchmark data
addpath('src');
cd examples
run generate_NLS_data.m    % saves NLS_data.mat

% 2. Run the TASC-DMD benchmark (reproduces Fig. 2)
run Schrodinger_equ_TASCDMD.m
